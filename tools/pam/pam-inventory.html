SÉRIE 4 · PAM · ISO 8.2 · SC-300

PAM Inventory

Cataloga e classifica contas privilegiadas do Microsoft Entra ID em tiers de acesso (Tier 0–3), identifica contas sem controle PAM adequado e gera inventário alinhado ao controle ISO/IEC 27001:2022 — 8.2 (Privileged Access Rights).

Modelo de tiers PAM — referência

Tier 0 — Controle do tenant

Global Admin, Privileged Role Admin. Acesso total ao plano de controle. Exige conta dedicada, MFA obrigatório, PIM, monitoramento contínuo.

Tier 1 — Administração de serviços

User Admin, Exchange Admin, Intune Admin. Controla serviços críticos. Exige MFA, acesso just-in-time via PIM.

Tier 2 — Acesso elevado específico

Security Reader, Helpdesk Admin, grupos críticos. Acesso privilegiado com escopo limitado. MFA recomendado.

Tier 3 — Acesso padrão

Roles de leitura, grupos de negócio. Sem privilégio administrativo direto. Revisão periódica.
