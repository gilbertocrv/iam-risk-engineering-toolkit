Conditional Access Analyzer

Inventaria e analisa políticas de Conditional Access do Microsoft Entra ID. Detecta lacunas de cobertura, políticas sem ação definida, ausência de MFA em grupos críticos e documenta atributos ABAC utilizados.

Referência — atributos ABAC no Entra ID
IDENTIDADE
Usuário · Grupo · Role de diretório · Conta de serviço
LOCALIZAÇÃO
Named Location · País · IP · Rede corporativa
DISPOSITIVO
Compliance (Intune) · Hybrid joined · Tipo · Plataforma
RISCO
Sign-in risk · User risk · Session risk (Identity Protection)
AUTENTICAÇÃO
MFA satisfeito · Authentication strength · SSPR
APLICAÇÃO
App específica · Todas as apps · Portais de admin
1b. Importar export JSON do Graph API (opcional)

Se preferir importar diretamente do Microsoft Graph, exporte as políticas via PowerShell e carregue o JSON aqui. As políticas serão parseadas automaticamente.

Connect-MgGraph -Scopes "Policy.Read.All" $policies = Get-MgIdentityConditionalAccessPolicy $policies | ConvertTo-Json -Depth 10 | Out-File "ca-policies.json"
Carregue o JSON exportado via Get-MgIdentityConditionalAccessPolicy.
